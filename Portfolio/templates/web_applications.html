Segmentation Model — Marketing Science Ltd

Django GPT-5 API Numpy Scikit-learn Statsmodels Html CSS Javascript

The Segmentation Model by Marketing Science Limited is a user-friendly web platform for analyzing behavioral data and generating AI-powered personas. Designed for marketing, research, and analytics, it transforms raw behavioral data into clear, actionable insights. The workflow includes data input, standardization, factor analysis, clustering, persona generation (via GPT-5 mini), and export of structured results directly to Microsoft Excel. It also provides robust visualization tools such as PCA plots and silhouette analysis.

Hosted at segmentation.marketsci.co.uk. Copyright © Marketing Science Ltd.

Restaurant Inventory Management System — Flask + PyQt5

A role-based inventory management system designed for restaurant operations. Built with Flask (backend) and PyQt5 (client GUI), it features secure account creation (passwords stored as hashes), role-based access (Head Chef, Chef, Rider), automated stock reordering, expiry date tracking, and real-time shortage alerts. The app launches in a mobile-sized window, designed for quick adoption in kitchen environments.

Interactive Menu & Checkout Website — Django

A dynamic restaurant website featuring a multilingual menu (English ↔ Arabic), seamless checkout via Stripe API, and cart management. Customers can browse items, add them to cart, and complete purchases securely. The platform was engineered with scalability and localization in mind, ensuring accessibility for diverse audiences.
