Anomaly-Based Intrusion Detection System

My final year group project for Bachelors in Cybersecurity — a full pipeline IDS leveraging Nvidia Morpheus, Docker, Apache Kafka, Filebeat, Logstash, and Grafana. The system ingests CloudTrail logs in real time, fingerprints user activity, and flags anomalies through an Autoencoder-based model, with interactive dashboards for monitoring.

System Pipeline

CloudTrail Logs
Filebeat / Logstash
Apache Kafka
Nvidia Morpheus
Grafana Dashboards

End-to-end flow: CloudTrail logs → Filebeat/Logstash for preprocessing → Apache Kafka (Dockerized) → Nvidia Morpheus pipeline → Autoencoder model for anomaly scoring → Grafana dashboards for visualization.

Project Overview

This system was built to demonstrate how modern data engineering tools can be fused with AI models to detect security anomalies in cloud environments. Key highlights include:

  • Scalable Ingestion: Logs streamed into Kafka topics running inside Docker containers.
  • Deep Learning Detection: Morpheus pipeline with an Autoencoder trained on digital fingerprints of users.
  • Real-time Dashboards: Grafana visualizes anomalies, trends, and live metrics.
  • Cloud-ready: Compatible with AWS/Azure CloudTrail logs.

System Architecture & Technical Implementation

AWS CloudTrail Ingestion

Collects JSON-formatted audit logs containing user activities, API calls, and security events from AWS environments for real-time analysis.

Log Processing Pipeline

Filebeat ships logs while Logstash performs ETL operations using Grok patterns for parsing, filtering, and data normalization.

Stream Processing

Dockerized Kafka cluster provides fault-tolerant message streaming with configurable partitioning for high-throughput data processing.

GPU-Accelerated Pipeline

Nvidia Morpheus framework with RAPIDS cuDF orchestrates ML pipeline stages for feature engineering and real-time inference.

Anomaly Detection Model

Deep learning Autoencoder trained on user behavior patterns, using reconstruction error as anomaly scores for threat detection.

Real-time Monitoring

Grafana dashboards with Prometheus integration provide real-time visualization of anomaly scores and system metrics.

Resources & References: Development leveraged Nvidia's self-paced Morpheus courses and official Nvidia Morpheus GitHub repository for implementation guidance and best practices.
